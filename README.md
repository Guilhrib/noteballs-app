# noteballs-app
Application developed during Danny Cornell's Vue 3 course in order to study the main differences between the Options API and the Composition API.

## Getting Started

1. To install the dependencies
````````````````````````````````````````````````````````````````````````````````````````````````````````````````````
    npm install
````````````````````````````````````````````````````````````````````````````````````````````````````````````````````
2. To start the application
````````````````````````````````````````````````````````````````````````````````````````````````````````````````````
    npm run dev
````````````````````````````````````````````````````````````````````````````````````````````````````````````````````
